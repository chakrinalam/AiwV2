# 🌱 AllIsWell NGO Website

A modern and responsive website developed for the **AllIsWell NGO** to showcase its mission, initiatives, and community activities.  
The platform provides visitors with information about the organization, its services, and ways to engage with or support its initiatives.

This project focuses on delivering a clean, user-friendly interface while ensuring responsiveness across devices.

---

# 📌 Project Overview

The **AllIsWell NGO Website** is designed to serve as the digital presence of the organization.  
It provides essential information about the NGO, its activities, and opportunities for people to participate in or support its initiatives.

The website aims to:

- Increase awareness about the NGO  
- Provide easy access to information about services and initiatives  
- Enable community engagement  
- Provide utility tools such as a **Coordinate Finder**  
- Display official **Certificates and recognitions** related to the organization  
- Prepare the foundation for future donation functionality  

---

# ✨ Features

- Responsive and modern UI design
- Informational sections about the NGO and its mission
- Services and initiatives showcase
- Volunteer and engagement sections
- Contact information for reaching the organization
- Dedicated **Donate page (UI implementation)**
- **Coordinate Finder** – Allows users to find geographical coordinates for a specific location
- **Certificate Page** – Displays certificates and recognitions related to the NGO and its activities

⚠️ **Note:**  
The donation payment gateway integration (e.g., Razorpay) is **not yet implemented**. The donate page currently provides only the interface.
